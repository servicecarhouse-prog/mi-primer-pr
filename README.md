# mi-primer-pr

Este es mi primer repositorio en GitHub.

## Descripción

Un proyecto de prueba para aprender el flujo de trabajo con Git y GitHub:
crear ramas, hacer commits y abrir Pull Requests.

## Pasos del flujo de trabajo

1. Clonar el repositorio
2. Crear una rama nueva
3. Hacer cambios
4. Abrir un Pull Request
5. Fusionar los cambios

## Requisitos

- Git instalado
- Cuenta de GitHub
- GitHub CLI (`gh`)
